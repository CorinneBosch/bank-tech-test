## Setup

1. **Fork** this []()
2. Then clone **your** fork to your computer.
3. Ensure you have `Ruby`, `Rails` and `rspec` installed.
4. Run `gem install bundler` if you dont have bundler.
5. Once installed, run `bundle`.
